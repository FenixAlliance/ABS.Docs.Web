TimeSpan NuGetGallery.ODataCacheOutputAttribute.ReloadDuration = TimeSpan.FromSeconds(60)
getset

This setting determines how frequently the cache duration setting should be checked. Since the OData endpoints get a lot of traffic, we don't want to check every time. The 60 seconds here is picked to mimic the feature flag reload time and the value has nothing to do with the cache durations themselves.
