Task<IPLookup> FenixAlliance.ABM.Data.Access.Services.IpLookupService.GetIpLookup ( HttpContext  httpContext)

If a User is not authenticated, it will create a new IP Lookup only if no IP lookup for the current remote IP has been found.

If a User is authenticated, and no Owned Lookup for either this IP or this device has been found, it will create a new Owned IP Lookup linked to that User and send an alert email.
