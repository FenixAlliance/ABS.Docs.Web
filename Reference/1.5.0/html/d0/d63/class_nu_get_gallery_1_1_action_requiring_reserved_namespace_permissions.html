Detailed Description

An action requiring permissions on ReservedNamespaces that can be done on behalf of another User.

These permissions refer to IReadOnlyCollection<ReservedNamespace> and not a single ReservedNamespace because multiple namespaces can apply to a single ID. E.g. "JQuery.Extensions.MyCoolExtension" matches both "JQuery.*" and "JQuery.Extensions.*".
