◆ PackageVersionRange

string NuGet.Services.Entities.VulnerablePackageVersionRange.PackageVersionRange

The version range of PackageId that is vulnerable.

The version range with the maximum possible length has:

  • both a minimum and maximum version with the maximum version length (Constants.MaxPackageVersionLength * 2)
  • two symbols on both sides for inclusive/exclusive (2)
  • the comma and space between the minimum and maximum versions (2) For a total of Constants.MaxPackageVersionLength * 2 + 4.
