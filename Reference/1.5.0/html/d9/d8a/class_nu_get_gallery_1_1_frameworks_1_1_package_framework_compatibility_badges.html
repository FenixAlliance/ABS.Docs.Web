Detailed Description

Contains a NuGetFramework for each of the .NET product (.NET, .NET Core, .NET Standard, and .NET Framework).

All these properties are retrieved from the PackageFrameworkCompatibility.Table, one for each .NET product.
Only package asset frameworks are considered. i.e. PackageFrameworkCompatibilityTableData.IsComputed = false.
If there are no package asset frameworks for a particular .NET product, then the value will be null.
