Tries to set the instantiated content 's Aseembly.

  1. First, it will try to search for an instantiated assembly on cache. If found, will set the content 's Assembly and return true.
  1. It will try to go to the File System to see if a precompiled dll is present. of so, will instantiate and set the content 's Assembly and return true.
