async Task<Order> FenixAlliance.ABM.Data.Access.Services.OrdersService.CreateOrder ( AccountHolder  holder,
Cart  cart,
SalesOrder  order = null 
)

Creates an SalesOrder owned by the passed AccountHolder as well as the Cart owner.

Orders that contains products from multiple business tenants will be created as composite orders, meaning that the system will first group each ItemCartRecord by their owner business and then create an order for each business tenants. This order will be bound to a global parent order.

If Account Holder does not have permissions to create orders for the passed cart, a new UnauthorizedAccessException will be thrown.

Parameters
holderThe owner for this order.
cartThe cart from which a list of ItemCartRecord will be extracted to create InvoiceItemRecord.
orderThe order tu fullfil. if null or not passed, a new SalesOrder weill be created.
Returns
A new SalesOrder already created into the database.
Exceptions
UnauthorizedAccessException
