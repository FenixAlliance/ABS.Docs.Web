async Task<DeleteAccountStatus> NuGetGallery.DeleteAccountService.DeleteAccountAsync ( User  userToBeDeleted,
User  userToExecuteTheDelete,
AccountDeletionOrphanPackagePolicy  orphanPackagePolicy = AccountDeletionOrphanPackagePolicy.DoNotAllowOrphans 
)

Will clean-up the data related with an user account. The result will be:

  1. The user will be removed as owner from its owned packages.
  2. Any of the packages that become orphaned as its result will be handled according to orphanPackagePolicy .
  3. Any owned namespaces will be released.
  4. The user credentials will be cleaned.
  5. The user data will be cleaned.
Parameters
userToBeDeletedThe user to be deleted.
userToExecuteTheDeleteThe user deleting the account.
orphanPackagePolicyIf deleting the account creates any orphaned packages, a AccountDeletionOrphanPackagePolicy that describes how those orphans should be handled.
