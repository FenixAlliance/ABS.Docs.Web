Detailed Description

Processes templates by running a series of string.Replace(string, string) calls against a template. Not very efficient but simple.

Template Parameters
TInputThe data source type for placeholder substitution.

Known issue:

The order in which string.Replace(string, string) calls are performed is undefined, so if substitution results in producing some other placeholder, the result would depend on order of operation and thus be undefined as well. Need to keep that in mind when picking the placeholder representation and consider what they can be substituted with.
