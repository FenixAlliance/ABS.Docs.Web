Detailed Description

This class represents a direct relation between an Account Holder (IdentityUser) to a Business Tenant (Business).

A present record of this type basically allows fine-grained controlled access to certain core Business functionalities like the hability to act on behalf of a business given certain roles and direct access permissions.

Certain entities, (like Contacts, Invoices, Items, Agreements and more) require a reference to this type to be present either for relational DB purposes and UI Generation as it represents the "Creator Record" (BusinessProfileRecordID).

Every Business Tenant contains a Root Business Profile Record that stablishes a link between the Account Holder that installed the system and the Root Business Tenant.

In cases where a BusinessProfileRecordID is not provided to a record, the system will asign the Root Record unless otherwhise specified on the application configuration.
