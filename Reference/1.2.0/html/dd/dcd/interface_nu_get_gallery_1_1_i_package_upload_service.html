Commit the provided package metadata and stream to the package file storage and to the database. This method commits the shared IEntitiesContext. This method can throw exceptions in exceptional cases (such as database failures). This method does not dispose the provided stream but does read it.

Parameters
packageThe package metadata. This is assumed to already be added to the context.
packageFileThe seekable stream containing the package content (.nupkg).

Validate the provided package once the new packages owner is known. The validations performed by this method should make sense for both the UI upload and command line push. This should be called after GeneratePackageAsync(string, PackageArchiveReader, PackageStreamMetadata, User, User) but before CommitPackageAsync(Package, Stream).

Parameters
packageThe package entity not yet committed to the database.
nuGetPackageThe package archive reader.
ownerThe owner of the package.
currentUserThe current user.
isNewPackageRegistrationDetermine whether the uploaded package is a new package without existing package registration info.
Returns
The package validation result.
