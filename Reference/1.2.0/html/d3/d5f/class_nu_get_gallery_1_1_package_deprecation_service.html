Updates the deprecation of many packages. If any packages have an existing deprecation, it combines the existing state with the new state. If only a single package is provided, the existing deprecation is entirely replaced. If the deprecation supplied is not vulnerable, not legacy, and not other, the deprecations of the packages will be removed. Commits changes when finished.
