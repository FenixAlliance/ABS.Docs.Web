◆ PermissionsRequirement

enum NuGetGallery.PermissionsRequirement
strong

Represents a permissions restriction applied to a User who is trying to do an operation User has with a resource such as a Package or account (e.g. another User).

Enumerator
Unsatisfiable 

No user can satisfy the requirement.

None 

No requirement–any user can satisfy the requirement.

Owner 

If the user is an owner of the Package, User, or ReservedNamespace, they satisfy the requirement.

SiteAdmin 

If the user is an admin of the site (see GalleryConstants.AdminRoleName), they satisfy the requirement.

OrganizationAdmin 

If the user is an admin of an Organization (Membership.IsAdmin is true) that satisfies Owner, they satisfy the requirement.

OrganizationCollaborator 

If the user is a collaborator of an Organization (Membership.IsAdmin is false) that satisfies Owner, they satisfy the requirement.
